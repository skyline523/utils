# Get Started

This is my commonly used tool chain, which is stable and continuously maintained and updated.

## Installation

::: code-group

```bash {npm}
npm i @xperia11/utils
```

```bash {yarn}
yarn add @xperia11/utils
```

```bash {pnpm}
pnpm add @xperia11/utils
```

:::

## Usage

```js
import { isMobile } from '@xperia11/utils'

if (isMobile()) {
  // do something...
}
```
